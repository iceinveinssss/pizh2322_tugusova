## Задания  

### Класс `Roman`  
Создайте класс `Roman` (РимскоеЧисло), который представляет римское число и поддерживает операции +, -, *, /. Реализуйте операции +, -, *, / как специальные методы (add и др.). Также реализуйте статические методы для преобразования арабских чисел в римские и римских чисел в арабские.

### Генераторное выражение  
Создайте генераторное выражение, которое из списка чисел `numbers` возводит в квадрат те числа, которые делятся на 3. Затем распечатайте сумму этих квадратов.

Пример:

Для списка `numbers = [1, 3, 4, 6, 9, 11]` программа должна вывести сумму квадратов чисел 3, 6 и 9, то есть:
\[
3^2 + 6^2 + 9^2 = 126
\]

Для списка `numbers = [1, 4, 11]` программа должна вывести 0.

### Декоратор `format_float_return()`  
Напишите декоратор `format_float_return()`. Этот декоратор должен проверять значение, возвращаемое задекорированной функцией. Если возвращается значение типа `float`, декоратор должен округлить его до двух знаков после запятой. Если возвращаемое значение другого типа, декоратор не должен изменять результат.

Для округления используйте функцию `round()`.
