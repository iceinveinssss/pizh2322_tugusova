## Задания
Класс `Pizza`
Создайте базовый класс Pizza, который представляет пиццу. Класс должен содержать атрибуты: название пиццы, тип теста, тип соуса и список начинок. Реализуйте методы для подготовки, выпечки, нарезки и упаковки пиццы.

### На основе базового класса создайте три дочерних класса:

PepperoniPizza — пицца Пепперони.

BarbecuePizza — пицца Барбекю.

SeafoodPizza — пицца Дары Моря.

Каждый из этих классов должен инициализировать атрибуты базового класса с соответствующими значениями.

Класс Order
Создайте класс Order, который представляет заказ. Класс должен содержать список пицц, добавленных в заказ, и методы для добавления пиццы, расчета общей стоимости и отображения информации о заказе.

Класс Terminal
Создайте класс Terminal, который отвечает за взаимодействие с пользователем. Класс должен содержать меню пицц, методы для отображения меню, приема заказа, обработки оплаты и завершения заказа.
